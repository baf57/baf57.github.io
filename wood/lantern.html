Lantern

This project consisted of remaking the backing board for a lantern. This was my first commissioned project.

The Lantern came as a kit with a very cheap composite backing board. I used a piece of live edge walnut to replace the backing board.

Since I was given this kit by the client, I had to use some of the original backing board to achieve the proper backing board thickness. To deal with this I made sure that the back board was is mostly hidden or unnoticeable from any of the main viewing angles.

This project did not take very long, only about 5 hours. This project was mostly designed by the customer, while finalized and completed by me. I will be installing it in their home when they are available some time in early October.
