GitHub Projects

Universal Electric Potential Field Solver

A Python program made using the interactive Jupyter environment. The program is a 2 dimensional electric potential field solver which allows for the approximation of an electric field given regions of charge. It was extended to also allow for simulations of point charges within the field. Extensive documentation is provided within the Jupyter notebook, as the project was done for a computational physics course. It is important to note that the notebook will not compile correctly within GitHub and must be run locally for full functionality.

config-log

A simple shell script which generates a centralized log of changes made to any files edited using this script. The original design of this was for use in tracking changes to Unix configuration files.

mass-rename

A simple shell script which allows for the batch renaming of files in a directory based on globing. Was done as both a coding exercise as well as for my own use. Similar to the package rename, but without the direct use of Perl expressions and with simpler syntax.
